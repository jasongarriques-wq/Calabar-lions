# Static assets

Drop the following two image files into this directory and push them
(they're referenced by the landing page and the navbar):

| Filename           | What it is                                  |
| ------------------ | ------------------------------------------- |
| `lion-shield.png`  | The green Calabar Lions shield logo (used in the navbar and footer; small transparent-bg PNG is ideal) |
| `lion-pride.jpg`   | The lion-and-cubs scene (used as the hero background on the landing page; high-resolution landscape JPG is ideal) |

The pages will render without these files but the visuals depend on them.
